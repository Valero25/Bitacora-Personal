# Bitácora Personal

Esta es mi bitácora personal para practicar y mejorar en programación. Contiene carpetas organizadas por semanas (semana 1 a semana 16), donde subiré mini ejercicios de práctica para mostrar al profesor y seguir avanzando.

## Estructura
- **semana1/** a **semana16/**: Carpetas para cada semana de práctica. Cada una incluye un README.md con evidencia técnica de los ejercicios resueltos (clase y casa).

## Cómo usar
- Agrego mis ejercicios en la carpeta correspondiente a la semana.
- Uso GitFlow: cada semana creo una rama `feature/semanaX` desde `develop`, trabajo en ella, y la mezclo de vuelta a `develop`.
- Documento la evidencia en el README semanal.
- Hago commits regulares para rastrear mi progreso.

## Especificaciones
- **Código en repositorio Git**: Todo mi código se almacena aquí.
- **Ejercicios resueltos**: Incluyo ejercicios de clase y casa, con evidencia en el README de cada semana.
- **GitFlow**: Uso ramas feature para cada semana, mezcladas en `develop`.
