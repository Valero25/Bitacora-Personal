# Bitácora Personal

Esta es mi bitácora personal para practicar y mejorar en programación. Es un proyecto Maven que contiene carpetas organizadas por semanas bajo `src/main/java/dosw/bitacora/` (desde semana 1 hasta semana 16 inicialmente, pero por ahora hasta semana 3), donde subiré mini ejercicios de práctica para mostrar al profesor y seguir avanzando.

## Estructura
- **src/main/java/dosw/bitacora/semana1/** a **semana3/**: Carpetas para cada semana de práctica. Cada una incluye un README.md con evidencia técnica de los ejercicios resueltos (clase y casa), y subcarpetas para temas específicos (ej. streams en semana1, solid y patrones en semana2).

## Cómo usar
- Agrega tus ejercicios en la carpeta correspondiente a la semana.
- Usa GitFlow: cada semana crea una rama `feature/semanaX` desde `develop`, trabaja en ella, y mézclala de vuelta a `develop`.
- Documenta la evidencia en el README semanal.
- Haz commits regulares para rastrear el progreso.

## Especificaciones
- **Código en repositorio Git**: Todo el código se almacena aquí.
- **Ejercicios resueltos**: Incluyen ejercicios de clase y casa, con evidencia en el README de cada semana.
- **GitFlow**: Usa ramas feature para cada semana, mezcladas en `develop`.
